## 💡 Troubleshooting

If you encounter issues, review this checklist.

- The trainer does not activate: Check that you started the game before the trainer. Some features require an active game instance to find the memory addresses.
- The game crashes: Ensure you are not using other software that modifies game memory at the same time. Conflict between different trainers often causes instability. 
- Options do not work: Verify that the version of Hades II you play matches the version the trainer supports. Updates to the game can shift memory addresses, requiring a trainer update.
- Performance drops: If your game lags, toggle off features like Rapid Dash to lower the load on your system.
